# Generated Knowledge Snapshots

This directory contains generated or generator-owned snapshots that make the
repo easier for agents to inspect.

Current files:

- `block-types.md` — generated inventory of built-in Block Type source files.
- `std-registry.md` — lightweight snapshot of the public `#__std` registry shape.

Regenerate or update these files when the source they summarize changes. The
runtime source of truth remains the Nix source and `#__std` output.
